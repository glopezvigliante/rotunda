# Rotunda Exercises

## Requirements

[Node.js](https://nodejs.org/) version [v10](https://nodejs.org/download/release/latest-v10.x/) or superior.

## Execution

### URL Parser

```bash
node url-parser '/:version/api/:collection/:id' '/6/api/listings/3?sort=desc&limit=10'
```

### Error Alarm

```bash
node error-alarm
```

### Zoo

```bash
node zoo
```
